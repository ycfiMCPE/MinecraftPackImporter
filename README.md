# MC Pack Importer

A simple tool to bulk import `.mcpack` and `.mcaddon` files into Minecraft Bedrock Edition — no more importing one by one through the game.

**by zzzzzzzzzvzzzzzzzzz · [discord.gg/hivebw](https://discord.gg/hivebw)**

---

## Features

- Drag & drop packs directly onto the app
- Import multiple packs at once
- Auto-detects resource packs, behavior packs, and skin packs
- Supports `.mcpack` and `.mcaddon` files
- Restart Minecraft directly from the app
- Change your Minecraft folder path in Settings

---

## How to Use

1. Download `MinecraftPackImporter.exe` from the [Releases](../../releases/latest) page
2. Drag your `.mcpack` / `.mcaddon` files onto the app — or click to browse
3. Hit **Restart MC** after importing so packs show up in-game

> **Note:** Minecraft must be restarted after importing packs.

---

## Wrong Minecraft Folder?

Go to **Settings** tab and set your `com.mojang` folder path. It saves automatically.

Default path:
```
%LOCALAPPDATA%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang
```
